# Birthday Manager App

## Overview
This Python application allows users to manage birthday records. It allows users to add, search, update, and delete birthday information stored in a dictionary.

## Features
- Look up birthdays
- Add new birthday records
- Update existing birthdays
- Delete birthday records

## Technologies Used
- Python

## How It Works
1. Users select options from the menu.
2. Birthday records are stored using a dictionary.
3. Users can:
   - Search for birthdays
   - Add new birthdays
   - Update existing birthdays
   - Delete birthdays
